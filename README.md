# Topic Chooser
This is a tool to choose one conversation topic from http://iteslj.org/questions/, which is assumed to be used for English speaking lessons.

![ss01](https://github.com/wtetsu/topic-chooser/blob/images/ss01.png)

## Build
```tty
yarn install
yarn start
```
